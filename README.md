Расширение выпелено из всех магазинов браузеров, поэтому оно и находится здесь.

Как включить расширение?

Для Windows + Google Chrome
1. Скачать архив SkyloadExtension.zip из данного репазитория, и распаковать в любое место (условно в папку SkyloadExtension)
2. Нажать правой кнопкой мыши на значке браузера Google Chrome и выбрать "Свойства"
3. По умолчанию открыта вкладка "Ярлык"
В поле "Объект" добавьте после пробела:--disable-features=ExtensionManifestV2Unsupported,ExtensionManifestV2Disabled
чтобы получилсь вот такая строка по итогу

"C:\Program Files\Google\Chrome\Application\chrome.exe" --disable-features=ExtensionManifestV2Unsupported,ExtensionManifestV2Disabled

4. Применить/Ок

5. Открываем браузер по этому ярлыку, переходим в Настройки/Расширения
chrome://extensions/

6. Включаем режим разработчика (Ползунок в правом верхнем углу), появляются доп кнопки

7. Загрузить распакованное расширение

8. Выбрать папку SkyloadExtension с раширением и подтвердить.

Готово!


Для браузера от яндекса похожая инструкция, за исключением 5 пункта, придётся самому ввести в адресной строке браузера browser://extensions/
